# Portfolio

This project is a React-based portfolio app powered by [Vite](https://vite.dev/).

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

You can also use:

```bash
npm start
```

By default, Vite serves the app at `http://localhost:5173`.

## Available Scripts

### `npm run dev`

Starts the Vite development server with hot reload.

### `npm start`

Alias for `npm run dev`.

### `npm run build`

Builds the app for production into the `dist` folder.

### `npm run preview`

Serves the production build locally for verification.

### `npm test`

Runs tests using Vitest (configured to pass when no test files exist).

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Sass
- Framer Motion
